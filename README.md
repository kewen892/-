# -
漫剧自动化生成工具 - AI驱动的内容生产流水线
🎬 manju-Tool · 漫剧自动化生成工具
AI驱动的漫剧创作流水线管理平台。

核心功能
📜 剧本解析
上传简短故事→AI扩写并生成剧本 → AI自动提取角色、场景、道具等元素→AI自动生成资产包→AI根据剧本和提取的元素自动生成分镜首帧图和分镜视频→AI打包分镜视频并智能连贯合并视频

👤 角色管理
支持角色多版本管理（不同时期/服装/武器/配饰）
每个版本独立生成图片
角色与分镜关联
🏞️ 场景管理
场景类型标签（室内/室外/幻想/历史）
时间设定（白天/夜晚/黎明/黄昏）
场景与分镜关联
🎒 道具管理
道具分类（武器/服装/配饰/工具）
道具与分镜关联
🎞️ 分镜制作
AI自动从剧本生成分镜
关联资产：自动带入角色+场景+道具+配饰
支持批量生成提示词
🤖 AI集成

可对接各大AI平台的API，也可以本地部署AI大模型后调用本地API（由于该工具工作量较大，本地部署AI大模型的硬件要求较高，若嫌各大AI服务商的API费用较高的话可以在云端部署大模型，目前阿里云百炼就很不错）
使用方法
直接在浏览器打开 index.html 即可使用（本地版，仅测试用）。

或者部署到云服务器在线访问。
